## ① Supabase マイグレーション適用（天気カテゴリ/テンプレ＝0027 を本番へ）
```bash
cd supabase
npx supabase link --project-ref lxkvzofgzkksujoipnso   # 未linkなら
npx supabase db push
```
- これで `0027_weather_templates`（天気カテゴリ＋デイリーテンプレ）が本番に入る。
- ※ 既に手動でweather/entカテゴリとデモ市場を入れてあるので、0027は `on conflict` で衝突せず冪等。

## ② Edge Functions デプロイ（天気の自動生成・自動解決を有効化）
```bash
npx supabase functions deploy generate-markets
npx supabase functions deploy resolve-markets
```
- `generate-markets`（15分cron）＝天気テンプレから「今日/明日の東京は雨？」等を自動生成
- `resolve-markets`（5分cron）＝対象日翌日にAMeDAS観測で自動解決
- Functionの環境変数 `SUPABASE_URL` / `SUPABASE_SERVICE_ROLE_KEY` は Supabase が自動注入（天気はキー不要）。

## ③ cron 確認（自動実行されているか）
- Supabase ダッシュボード → Database → Cron（pg_cron）で `generate-markets` / `resolve-markets` のジョブが有効か確認。
- 無ければ migration 0009 のcron登録を確認（既存）。

## ④ LINEログイン チャネル作成（LINE Developers）
1. https://developers.line.biz/ でプロバイダー作成 → **LINE Login** チャネルを新規作成
2. **Channel ID** と **Channel Secret** を控える
3. **コールバックURL**を登録（完全一致が必須）:
   ```
   https://<本番ドメイン>/api/auth/line/callback
   例: https://d-market.io/api/auth/line/callback
   ```
4. スコープ: `profile` `openid`（コードが要求済み）

## ⑤ Vercel 環境変数を設定（Production）
Vercel → Project → Settings → Environment Variables に追加:
| 変数 | 値 |
|---|---|
| `LINE_LOGIN_CHANNEL_ID` | LINEのChannel ID |
| `LINE_LOGIN_CHANNEL_SECRET` | LINEのChannel Secret |
| `LINE_AUTH_SECRET` | 任意のランダム長文字列（セッション用署名） |
| `NEXT_PUBLIC_SITE_URL` | `https://<本番ドメイン>`（末尾スラッシュ無し・LINE callbackと一致） |
| `SUPABASE_SERVICE_ROLE_KEY` | service role key（**NEXT_PUBLIC を付けない**） |
| `NEXT_PUBLIC_SUPABASE_URL` | （既存） |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | （既存） |
- 設定後、**再デプロイ**して反映。

## ⑥ Supabase Auth 設定の確認
- Authentication → Providers → **Email を有効**（LINEログインは内部でemail/passwordを使うため）。
- Email confirmations は callback側で `email_confirm: true` 指定済みなので確認メールは不要。

1. **LINEログイン**: トップの「LINEでログイン」→ LINE認可 → 戻ってログイン状態（残高ピル表示）。失敗時はURLの `?login=xxx&d=...` で失敗箇所が分かる（state/token/profile/create/signin/signup/config）。
2. **コメント**: ログイン後、市場詳細のコメントタブで投稿・いいねできる。
3. **取引**: 市場で「乗る」→ 参加ポイントで売買できる。
4. **天気の自動解決**: 7/3・7/5の天気デモ市場が、翌日にAMeDAS観測で自動確定するか（②デプロイ後）。
