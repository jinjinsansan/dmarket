# ゴリラ予想 リニューアル計画

> D-market を「ゴリラ予想」へリブランドし、白基調ミニマル＋ポップなデザイン、初心者向けの面白い市場、
> バイラル報酬・コメント・ユーザー作成市場（審査制）を追加する全体計画。
> 作成: 2026-06-29 ／ ステータス: 計画（着手＝デザイン刷新から）
> 関連: `Dmarket_PolyUIUX改善案.md` / `二層ポイント制_実装記録.md` / `参加ポイント獲得_アフィリエイト設計案.md` / `POYP実態調査_2026-06-29.md`

## 0. 確定した方針（2026-06-29 仁さん判断）

1. **報酬・テラ銭は全て「参加ポイント建て・換金不可」で統一**（賭博非該当の生命線）。新報酬（シェア/紹介/乗っかり1%）も市場作成者のテラ銭も、価値ゼロの参加ポイントで付与。
2. **「ゴリラ予想」へリブランド＋新ドメインを取得して移行**（現 `d-market.io` から）。
3. **ユーザー作成市場は審査制＋管理者が最終解決**（結果確定は管理者/オラクル。不正・係争を防ぐ）。
4. **着手はデザイン刷新から**（デザインブリーフ → Claudeデザイン）。

## 1. デザイン（①白基調ミニマル ③ポップ ⑦Xシェア最適化）

- 既存はデザイントークン（`web/app/globals.css` の `--bg`/`--surface`/`--primary`/`--pos`/`--neg`/`--radius` 等）を Tailwind v4 `@theme inline` に供給する作り。**変数名はそのまま、値だけ差し替えれば全体の見た目が変わる** → 改修コスト小。
- 詳細は `ゴリラ予想_デザインブリーフ.md`（Claudeデザインへ渡す）。
- 現状は青基調ライト（`--bg:#f4f8fd`）→ **純白〜オフホワイト基調＋1〜2色のポップアクセント＋ゴリラのマスコット**へ。
- ダークモードは既存（`html.dark`）を維持し、白基調に合わせて再調整。

---

## 2. 市場の中身（④初心者向け ⑥ユーザー作成）

### 2.1 初心者向けの面白い市場（④）
- 現 供給レイヤー（SPEC-04 `generate-markets`）は Polymarket を Gamma API でミラー。これを**テンプレ/キュレーション主導**へ転換（`market_templates` は既存）。
- 身近・エンタメ寄りのテーマ（芸能・スポーツ・天気・季節・身近なニュース）。YES/NO＋確率表示は維持しつつ、コピーと見せ方を初心者向けに。
- Polyミラーは縮小〜停止（`poly_max` を絞る）。

### 2.2 ユーザー作成市場＋テラ銭（⑥）★新規・要設計
- 基本は管理者が作成。**審査を通過したユーザーは市場を作成可能**。
- **作成者にテラ銭（控除分）が入る** ＝ **換金不可の参加ポイント建て**（方針0-1）。
- **結果確定（解決）は管理者/オラクルが行う**（方針0-3）。作成者は解決しない＝不正防止。
- 必要: ユーザー作成フロー＋審査ステータス（pending/approved/rejected）、作成者資格（`creator_approved`）、テラ銭の付与RPC（参加pt・reason 追加）、不適切テーマの通報/取り下げ、既存 `create_admin_market`/`admin_resolve` の拡張。
- 法務: 換金不可を厳守すれば賭博開帳図利の懸念を回避。審査＋管理者解決＋通報が運用ガードレール。

## 3. グロース/バイラル報酬（⑤a「貯める」ページ）★新規

すべて**参加ポイントの無償発行**（`point_ledger.reason` ホワイトリスト拡張＋RPC＋不在テスト更新）。二層ポイント制（0022-0026）の土台に乗る。

| 報酬 | 内容 | 備考 |
|---|---|---|
| ログインボーナス | 既存 `claim_daily_grant` を流用 | 実装済み |
| シェアボーナス | 自分の予想を X にシェアで付与 | シェア計測が必要 |
| 友達紹介ボーナス | 招待コード経由の新規登録で付与 | 紹介者・被紹介者の両取り設計 |
| 乗っかり＋的中1% | Xにシェアされた予想に別ユーザーが乗る → 元の市場が**的中したら、的中者の払い戻しの1%相当を参加ptでシェア元へ付与** | アトリビューション（誰のシェア経由か）テーブルが必要。付与は解決時フックで set-based |

- 既存オファーウォール（`参加ポイント獲得_アフィリエイト設計案.md`）と並ぶ「貯める」導線に統合。

## 4. コミュニティ（⑤b コメント機能）★新規

- 盛り上げUX: いいね、返信、（任意で）自分のポジション表示バッジ。

## 7. 法務ガードレール（横断・不変）

1. **新報酬・テラ銭は全て参加ポイント（換金不可）**。換金導線は一切作らない → 賭博・資金決済法リスクを生まない。
2. ユーザー作成市場は**審査＋管理者解決＋通報**。
3. シェア/紹介に有償アフィリ案件が絡む場合は**ステマ規制・特商法のPR表記**。
4. コメントは**通報/モデレーション**。
5. 公開前の**弁護士レビュー**（賭博・景表法・資金決済法・特商法・ステマ）。
